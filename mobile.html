How to Join on Mobile

Open Minecraft Bedrock Edition on your phone or tablet, add the server manually, and use the details exactly as shown here.

Server Address
nov-watches.gl.at.ply.gg
Port: 11439
  1. 1
    Open Play

    Launch Minecraft, tap `Play`, then go to the `Servers` tab.

  2. 2
    Add Server

    Tap `Add Server` and name it `GTSurvival` so it is easy to find later.

  3. 3
    Enter the Bedrock Address

    Paste `nov-watches.gl.at.ply.gg` and set the port to `11439`.

  4. 4
    Save and Join

    Tap `Save`, join the server, and head into survival with the rest of the community.

Mobile Player Tips

If the server does not appear right away, copy the address again and double-check the port.
